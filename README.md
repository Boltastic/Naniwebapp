# Nani — Web App Developer Portfolio

A premium futuristic portfolio website for showcasing AI-powered web apps, startup tools, and modern digital experiences.

Built with a cinematic light-theme aesthetic inspired by award-winning developer portfolios and creative studio websites.

---

## ✨ Features

* Ultra-modern responsive design
* Smooth Framer Motion animations
* Glassmorphism UI elements
* Cinematic scrolling experience
* Interactive project showcase
* Premium light theme aesthetic
* Mobile-first responsive layout
* Floating gradients and glow effects
* Clean developer-focused branding

## ⚡ Getting Started

Clone the repository:

```bash
git clone https://github.com/Boltastic/naniwebapp.git
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
